# vue-js-toggles

Vue toggle switches made with Vue and API Live demo: https://loving-panini-60f049.netlify.app/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
